How to run: from the dashboard folder, node serve.js 8790 (or python3 -m http.server 8790) → open http://localhost:8790/workbench.html. The app is static — no backend, nothing saved server-side. Opened directly from file://, it asks you to point at the data/ folder instead. Compiler rules implemented from benefits-hub-handoff/01-architecture/COMPILER_SPEC.md: PBP gating, content stack (base ← carrier ← workbench edits), slot resolution (null never renders as $0), §4a shared-wallet & VBID consolidation with the single-surface check, hard rules 1–2, admin-tool rows (§4b), EOC-vs-PBP discrepancy flags, D-SNP zero-dollar overlay, launch-scope stamping, and the per-plan report.
